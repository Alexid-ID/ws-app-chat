# Web demo WebSocket - WSChat
Real time chating simple demo
Run by nodejs and web socket

## Setting
```
yarn install
```
## Run project: 
```
yarn start
```
or
```
yarn run dev
```
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
